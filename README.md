# Afzal Portfolio

Welcome to my personal portfolio! This project showcases my skills, experience, and projects in data analytics, data science, and web development.

## Description

This portfolio is built with React and Vite, featuring a modern, cybernetic theme. It includes:
-   **Home**: Introduction and key statistics.
-   **About**: Details about my background and education.
-   **Work**: A showcase of my projects with links to live demos and GitHub repositories.
-   **Contact**: A way to get in touch with me.
-   **Services**: Services I offer.

The design emphasizes a futuristic aesthetic with dynamic animations and a responsive layout.

## Tech Stack

-   **Frontend**: React, Vite
-   **Styling**: Tailwind CSS, Vanilla CSS
-   **Language**: JavaScript
-   **Design**: Custom futuristic/cybernetic theme

## Getting Started

To run this project locally:

1.  Clone the repository:
    ```bash
    git clone https://github.com/afzals2004-cmyk/Portfolio.git
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    ```
